tinyAgent API

Welcome to the tinyAgent API! This is a simple yet powerful framework for building LLM-powered agents.

Use the endpoints below to interact with the API, or visit the API documentation for more details.

POST /chat

Chat with an agent using the built-in chat tool.

Parameters:

  • message: The message to send to the agent
  • session_id: A unique identifier for the chat session

POST /api/chat

Alternative endpoint for chatting with an agent.
